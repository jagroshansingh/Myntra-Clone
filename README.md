# Myntra-Clone
An E-commerce platform providing fashion and lifestyle solution on just a click.

It has following functionalities:
<ul>
  <li>Home Page with Navbar</li>
  <li>Project Page</li>
  <li>Wishlist Page</li>
  <li>Cart Page with the final purchase details</li>
</ul>

<h2> Tech Stacks used</h2>
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>
